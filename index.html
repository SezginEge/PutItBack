FAQ

The practical details.

This is the part search engines want and real users actually need: what the app is, what it does, and how the pricing and privacy story work in the current iOS build.

What does Put It Back actually do?

It is an iPhone focus timer built around phone pickup accountability. You choose a duration, start the session, put the phone down, and the app reacts if you pick it back up.

How does it know I picked up the phone?

The current iOS app requests motion access and uses motion data during active sessions to detect pickups. The pickup count feeds the escalation system and the session result.

What voice packs are included?

Mom is the free default pack. Sergeant and Therapist are paid packs that unlock together in a one-time purchase.

Does it track results or streaks?

Yes. The app stores session history, pickup counts, voice pack selection, clean-session streaks, and best streaks so the analytics and result screens can reflect what actually happened.

What data practices should I know about?

The current iOS code uses Firebase Analytics for usage events and StoreKit for purchases. Motion access is used for pickup detection, and session records are stored locally for in-app history and recovery.

Privacy and trust

Motion for focus. Not surveillance.

The supporting privacy page is built from the app's current behavior: motion access during sessions, local session storage, Firebase Analytics events, and App Store purchase handling for paid voice packs.
