FAQ

Practical details.

What is Put It Back?

Put It Back is an iPhone focus timer. You start a session, set the phone down, and the app tracks pickups during the timer.

How does it know I picked up the phone?

The current iOS app uses motion access during active sessions to detect pickups.

What is free and what is paid?

Mom is free. Sergeant and Therapist unlock together in a one-time purchase. No subscription is described in the current paywall.

What gets tracked?

The app stores session records, pickup counts, completion state, streak information, selected voice pack, and related session results.
